1. Css Syntax:
a Css Rule is made up of :
selector{
    property: value;
}

ex:
p{
    color: blue;
}

p-> the selector (targets <p> tag)
color-> the property
blue-> the value

2. Selectors explained:
selectors are used to target HTML elements to apply styles. Here are the most common:

A. universal selector *
*{
    margin: 0;
    padding: 0;
}
means : "apply this to every element on the page."

B. Type Selector(Tag selector)
h1{
    color: red;
}
means : "apply this to all <h1> elements"

c. Class selector .classname
.highlight{
    background-color: yellow;
}
means : "apply this to any elements with class =highlight"

used like this in HTML:
<p class ="highlight">THis is a highlighted paragraph.</p>

D. ID selector #idname

#special-box{
    border: 2px solid green;
}
means: "apply this to the element with id="special-box" "
used like this in HTML
<div id = "special-box">Content here</div>


remember:
class can be used on multiple elements
ID should be unique- used once per page

USE case REcap:
selector        target              syntax ex
*              all elements         *{ margin: 0; }
h1             all <h1> tags        h1{ color: red;}
.note          any element with     .note{ ... }
               class="note"
#important     element with id      #important{ ... }
                important

Tip: 
Combine selectors with styles logically to avoid repetition and keep your 
CSS clean and powerful
