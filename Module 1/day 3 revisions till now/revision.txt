Day 1: what is css?(inline, internal, external)
type            Applied in              use case example
inline          directly in HTML tag    <h1 style="color: red;">title</h1>
internal        <style> inside <head>   <style> p{ color : green;} </style>
external        Linked .css file        <link rel="stylesheet" href= "style.css">

day3: colors, units and comments
covered:
feature             example used
colors              rgb(), rgba(), hex, named
units               px, %, em, rem
comments            /* this is comments */

unit overview:
px: fixed pixels(not responsive)
%: relative to parent size
em: relative to parent element's font-size
rem: relative to root element's font-size
