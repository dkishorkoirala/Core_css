Day3: Colors, Units and comments
Goal:
-learn how to apply and customize Colors
-understand CSS units like px, %, em, and rem
- use comments to make your code clean and readable

key concepts
1. colors in CSS
you can define colors using:

color names(eg. red, blue)

hex codes(eg. #ff0000)

RGB values(eg. rgb(255, 0, 0))

RGBA values(eg. rgba(255, 0, 0, 0.5) for transparacy)

2. CSS units
units           Description
px              absolute pixels(fixed size)
%               relative to parent size
em               relative to font size of parent
rem              relative to root <html> font size


3. Comments in CSS
/*This is css comment*/
used to describe sections or temporatily disable code.
